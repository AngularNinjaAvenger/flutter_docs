WebSocketChannel class

A StreamChannel that communicates over a WebSocket.

This is implemented by classes that use dart:io and dart:html. The new WebSocketChannel constructor can also be used on any platform to connect to use the WebSocket protocol over a pre-existing channel.

All implementations emit WebSocketChannelExceptions. These exceptions wrap the native exception types where possible.
