hasFloatingPlaceholder property

bool hasFloatingPlaceholder
final

Whether the placeholder text floats to become a label on focus.

If this is false, the placeholder disappears when the input has focus or text has been entered. If this is true, the placeholder will rise to the top of the input when the input has focus or text has been entered.

Defaults to true.
