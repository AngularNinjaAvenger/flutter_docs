update method

Future<void> update (Uri golden, Uint8List imageBytes)

Updates the golden file identified by golden with imageBytes.

This will be invoked in lieu of compare when autoUpdateGoldenFiles is true (which gets set automatically by the test framework when the user runs flutter test --update-goldens).

The method by which golden is located and by which its bytes are written is left up to the implementation class.
