PercentDecoder class

A converter that decodes percent-encoded strings into byte arrays.

To be maximally flexible, this will decode any percent-encoded byte and will allow any non-percent-encoded byte other than %. By default, it interprets + as 0x2B rather than 0x20 as emitted by Uri.encodeQueryComponent.

This will throw a FormatException if the input string has an incomplete percent-encoding, or if it contains non-ASCII code units.
