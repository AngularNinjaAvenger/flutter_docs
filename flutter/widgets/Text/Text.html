Text constructor

const Text(String data, { Key key, TextStyle style, StrutStyle strutStyle, TextAlign textAlign, TextDirection textDirection, Locale locale, bool softWrap, TextOverflow overflow, double textScaleFactor, int maxLines, String semanticsLabel, TextWidthBasis textWidthBasis })

Creates a text widget.

If the style argument is null, the text will use the style from the closest enclosing DefaultTextStyle.

The data parameter must not be null.
