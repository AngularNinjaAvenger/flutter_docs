EdgeInsetsDirectional operator % (double other)
override

Computes the remainder in each dimension by the given factor.

Implementation

@override
EdgeInsetsDirectional operator %(double other) {
  return EdgeInsetsDirectional.fromSTEB(
    start % other,
    top % other,
    end % other,
    bottom % other,
  );
}
